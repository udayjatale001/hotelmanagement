# **App Name**: HarmonyHost App

## Core Features:

- Secure Admin Login: A dedicated login interface allowing access only with email 'suyash001@gmail.com' and password '9981' to ensure secure management.
- Food Token Generation: Module to generate unique serial numbered tokens for food items including Kachori, Samosa, Pohe, and Dhosa, with real-time storage.
- Inventory & Stock Tracking: Manage stock for items like Bisleri and Snacks, featuring automatic deduction of quantities upon sale and immediate display of updated stock levels.
- Billing & Invoice Generation: Generate itemized bills for customers, detailing services/items purchased and total amounts, all stored in real-time.
- Firebase Firestore Integration & Offline Persistence: Connects all data (Tokens, Stock, Sales, Bills) to Firebase Firestore Console, ensuring real-time cloud storage and enabling offline data entry with automatic syncing when internet connectivity is restored.
- AI-Powered Customer Note Tool: An AI tool that generates a brief, polite, and customized closing note for customer bills based on the order details, enhancing guest experience.

## Style Guidelines:

- Primary color: A professional and calming blue (#2273CC), signifying trust and efficiency for key interactive elements.
- Background color: A very light, almost white blue (#ECF1F5), offering a clean and unobtrusive canvas for content.
- Accent color: A fresh, analogous teal-blue (#46CCCC), used sparingly for highlights, alerts, and subtle distinctions.
- Font: 'Inter' (sans-serif) for all text elements. Its modern, objective aesthetic provides excellent readability for numerical data, lists, and general information on bills and management screens.
- Use clear, concise, and standard material design-style icons that are easily recognizable and reinforce functionality without visual clutter.
- Implement a clean, organized, and responsive layout that prioritizes information hierarchy and ease of navigation across different modules like inventory, billing, and token generation.
- Subtle and functional animations for feedback on user actions (e.g., token generation confirmation, stock updates) to enhance usability without being distracting.
